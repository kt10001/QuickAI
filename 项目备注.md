## 项目定位

one-api 的社区增强 Fork，更现代的UI + 更多功能，米酪等中转站使用的就是这个。

---

## 与 one-api 的核心差异

| 维度 | one-api | new-api |
|------|---------|---------|
| UI | 基础功能型 | 现代美观，多主题 |
| Midjourney | ❌ | ✅ 原生支持 |
| 充值/支付 | 基础 | 增强版，支付网关集成 |
| 更新频率 | 慢（个人维护） | 快（社区驱动，5490+ commits） |
| 渠道功能 | 基础 | 渠道重试/缓存/异步任务 |
| 多语言 | 中英 | 中/英/日/俄/越 |
| 数据统计 | 基础 | 更直观的仪表盘 |

---

## 部署命令

```bash
# Docker部署
docker run -d --name new-api \
  --platform linux/amd64 \
  --restart always \
  -p 3000:3000 \
  -e TZ=Asia/Shanghai \
  -v ~/data/new-api:/data \
  calciumion/new-api
```

默认账号：root / 123456
